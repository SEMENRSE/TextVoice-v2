# TextVoice v2 Beta

TextVoice v2 Beta is a Windows app for text-to-speech.

## Features

- Russian text-to-speech
- Russian voices: Irina and Ruslan
- English speech using Windows system voice
- TXT import
- TXT export
- Light and dark theme
- Automatic interface language: Russian / English

## Not included yet

- Microphone input
- Android version
- Built-in translator

## Download

Go to the latest release and download:
